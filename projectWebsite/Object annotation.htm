Object Annotation Task

Instructions

  1. You can see an image below with one object.
  2. Your task is to create the tightest bounding box around the main object in the image.
  3. For this, you can choose any shape from the options available and click on "Choose".
  4. Then draw a bounding box around the main object in the image using that shape and click on "Submit".
  • Shadows are not part of the image and should not be included in the bounding box
  • If you see a flower, do not include the stem of the flower in the bounding box

Here are some examples of the correct and wrong bounding box
